# pi-dask 术语表

## Language

**dask（结构化提问）**：
Pi 中用于收集可枚举用户选择的扩展与模型工具。它只负责单选/复选答案的交互和结构化返回，不决定问题逻辑、条件分支或决策结论。
_Avoid_: 问卷引擎、决策树、工作流

**平铺问题（Flat Questions）**：
一次 dask 调用中彼此不依赖、可一并收集的问题集合。任一题的答案不会改变另一题是否出现或其选项内容。
_Avoid_: 条件表单、跳题

**问题 ID（Question ID）**：
调用方为每道 dask 问题提供的稳定标识符，字段名为 `id`，用于将结构化答案与原问题对应。
_Avoid_: 题目序号、位置索引

**问题说明（Question Description）**：
与问题内容并列、由调用方提供的解释信息，用于说明该题要收集什么选择或回答时应考虑什么。它不决定问题顺序、推荐或条件分支。
_Avoid_: 决策逻辑、跳题规则

**题型（Question Type）**：
问题 `type` 字段的枚举值：`single` 表示单选，`multiple` 表示复选。它约束答案值数组的基数和 TUI 交互方式。
_Avoid_: 从选项数量推断题型

**选项 ID（Option ID）**：
调用方为某道问题的 `labels` 数组中每个枚举选项提供的用户可见序号，字段路径为 `question.labels[].id`。它必须与数组位置一致，从 `1` 连续递增到选项数；TUI（终端界面）将其显示在选项前，用户可在“其他（自行填写）”中用该编号指代选项。
_Avoid_: 任意稳定标识符、跳号、显示文案

**选项值（Option Value）**：
调用方在 `question.labels[]` 选项中声明、并在枚举选择后原样返回的 `value`。它仅可为字符串、数字或布尔值，且同一道题中的选项值必须唯一。
_Avoid_: 选项 ID、展示文案、对象、数组

**选项说明（Option Description）**：
与选项内容并列、由调用方提供的解释信息，说明该选项意味着什么、适合什么情况或会带来什么代价。它不构成 dask 的推荐或决策结论。
_Avoid_: 推荐理由、自动决策

**答案值（Answer Value）**：
答案中带来源的值对象，包含 `source` 与 `value`：`source` 为 `label` 时值来自枚举选项，为 `custom` 时值来自用户自行填写。每道题的 `value` 字段始终为此类对象组成的数组。
_Avoid_: 未标注来源的混合字符串、按题型变化的标量值

**答案列表（Answer List）**：
dask 返回的按提问顺序排列的答案对象集合；每个答案对象以问题 ID 与原问题关联。
_Avoid_: 按位置隐式关联、答案映射

**其他值（Custom Value）**：
用户通过每题固定的“其他（自行填写）”入口填写的补充答案。它以 `source: "custom"` 的答案值返回；在复选题中可与枚举选中项共存，但最多一项。
_Avoid_: 自由文本题
