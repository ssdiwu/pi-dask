# Changelog

本项目遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 的基本格式；版本以 Git tag（Git 标签）为准。

## [Unreleased]

## [0.0.4] - 2026-07-16

### Changed

- dask 工具调用和错误默认只显示摘要；请求参数与完整输出改为按 `Ctrl+O` 展开后查看。

## [0.0.3] - 2026-07-16

### Added

- 多题交互显示当前题、总题数和带已答/未答状态的题号栏，并支持数字键直跳及左右键相邻切题。

### Changed

- 允许先查看或回答任意题并保留各题草稿；摘要确认发现缺答时返回第一道未答题，不再生成不完整结果。
- npm（Node 包注册表）发布包包含完整 `doc/` 文档，确保 README 中的文档链接可用。

## [0.0.2] - 2026-07-16

### Changed

- 将模型可见的 `dask` 工具描述改为中文，并明确：真正需要用户决策且选项已收敛时，即使只有一道题也应使用工具，而不是在回复正文中列出编号选项；开放式探索及调用方应自行收口的工程细节不使用工具。
- 明确调用方应将当前已知且互不依赖的问题合并到一次 `dask` 调用；依赖其他答案的问题仍需分开调用。
- 选项 ID 改为从 `1` 开始、按展示顺序连续递增的用户可见序号；终端界面在每个选项前显示该序号，便于在“其他”答案中按编号指代选项。

## [0.0.1] - 2026-07-14

### Added

- 提供 `dask` 结构化提问工具，支持单选、复选、其他输入、回退、摘要确认和取消。
- 提供请求校验、接口契约和确定性测试。
